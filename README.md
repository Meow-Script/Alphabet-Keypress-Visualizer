# Alphabet Keypress Visualizer

An interactive Python application where keyboard inputs (A–Z) trigger custom images.

Each key press displays a visual representation associated with that letter.

## Features

- Event-driven keyboard input
- Custom image display for each letter
- Startup screen with start and exit functionality
- Interactive visual output

## Technologies

- Python

## How to Run

1. Install Python
2. Run the program:

python main.py
